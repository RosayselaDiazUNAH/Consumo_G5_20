20191003122 Cliente El formulario cliente es el que almacena los datos personales del cliente; en el se puede agregar, actualizar y eliminar cliente, estas acciones se realizan mediante botones creados, cada botón tiene el nombre de acuerdo a la acción a realizar.
Consta de 5 funciones llamadas:
1) CargarClientes: Cargar todos los clientes que estan en la base de datos, los muestra en la tabla.
2) AgregarCliente: Agregue clientes a la base de datos.
3) CargarCliente: Nos ayuda a la hora de actualizar y eliminar ya que carga un cliente de acuerdo a su llama primaria
4) ActualizarCliente: Actualiza los datos del cliente ya registrado en nuestra base de datos.
5) EliminarCliente: Borrar definitivamente un cliente, esta acción dr realiza mediante la llave primaria del cliente.

3) CargarCuenta: sirve para traer los datos de una sola cuenta, esto nos es de mucha utilidad al momento de actualizar
4) ActualizarCuenta: Actualiza los datos de una cuenta ya existente, cabe aclarar que numero de cuenta no se modificara.
5) EliminarCuenta: elimina por completo una cuenta que ya esta registrada en la base de datos.
